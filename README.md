## Signata Crypto

Signata Crypto is a YubiKey-based Cryptocurrency Wallet. You can create or import Bitcoin, Litecoin, Dash, Dogecoin, Ripple, and Ethereum addresses with your Private Keys encrypted by your YubiKey.
